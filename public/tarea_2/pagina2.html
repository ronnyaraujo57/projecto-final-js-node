Etiquetas, Descripcion y ejemplos

# Etiqueta Descripcion ejemplos
021 <blockquote> está pensada para identificar una cita larga, que puede contener varios párrafos u otras etiquetas.

022 <main> se utiliza para agrupar el contenido principal de una página, sin incluir los elementos comunes a otras páginas como bloques de navegación, información general del sitio, etc.

023 <div> define una división. Esta etiqueta permite agrupar varios elementos de bloque (párrafos, encabezados, listas, tablas, divisiones, etc).

024 <br> está pensada para insertar saltos de línea en un bloque (por ejemplo, para mostrar varios versos de un poema).

025 <a> hiperenlace

026 <span> no tiene un significado definido y es la que se debe utilizar cuando las otras etiquetas no son adecuadas. Se suele incluir el atributo class para especificar el significado que se le quiere dar a la etiqueta.

027 <strong> está pensada para indicar la importancia, seriedad o urgencia de su contenido.

028 <em> está pensada para dar énfasis a su contenido, es decir, para llamar la atención sobre ese contenido.

029 <sub> está pensada para identificar texto en subíndice.

030 <sup> está pensada para identificar texto en superíndice.

031 <abbr> está pensada para etiquetar abreviaturas o acrónimos. Si la etiqueta incluye el atributo title, este debe contener el significado de la abreviatura.

032 <data> está pensada para incluir en su atributo value la misma información que el texto, pero en un formato legible para máquinas.

033 <code> Pensada para etiquetar fragmento de código de ordenador.

034 <samp> Pensada para identificar un ejemplo de la salida de un programa o de un script.

035 <s> está pensada para indicar que se trata de información incorrecta o irrelevante.

036 <q> está pensada para identificar una cita o referencia a otras fuentes.

037 <dfn> está pensada para identificar dónde se define un término en un texto. La etiqueta abarca el término definido y su definición debe encontrarse en el mismo bloque (párrafo, elemento de lista, etc.)

038 <i> está pensada para identificar texto con un significado especial, como nombres de especies, términos técnicos, expresiones extranjeras, etc.

039 <b> está pensada para identificar texto sobre el que se quiere llamar la atención, pero sin que eso signifique que es importante, como los términos clave en un resumen, nombre de productos comerciales, etc.

040 <bdo> Pensada para elegir la dirección del texto (de izquierda a derecha o de derecha a izquierda). Es obligatorio especificar el atributo dir, con el valor rtl (de derecha a izquierda) o ltr (de izquierda a derecha).
