Etiquetas, Descripcion y ejemplos

# Etiqueta Descripcion ejemplos
061 <col> La etiqueta vacía <col> permite hacer referencia a una columna

062 <colgroup> permite definir grupos de columnas (de manera similar a como la etiqueta <tbody> define grupos de filas).

063 <form> es un conjunto de controles (botones, cajas de texto, casillas de verificación, botones radio, etc) que permiten al usuario introducir datos y enviarlos al servidor web para su procesamiento.

064 <input> Los controles son los elementos que permiten al usuario introducir información.
La mayoría de controles se obtienen mediante mediante la etiqueta vacía <input>.

065 <button> La diferencia entre ellos es que <input> sólo puede contener texto, mientras que <button> permite incluir elementos html como imágenes.

066 <label> permite asociar un control con un texto, de manera que mejore la accesibilidad de los formularios.
La asociación entre el control y la etiqueta <label> puede ser implícita o explícita.

067 <select> permite crear menús de opciones desplegable.

068 <option> permite agregar opciones a la etiqueta <select>

069 <optgroup> sirve para crear grupos de la etiqueta <option>

070 <fieldset> permite agrupar un conjunto de controles. Los navegadores muestran una caja alrededor de cada grupo de controles.

071 <legend> permite añadir una leyenda al <fieldset>. Los navegadores muestran la leyenda sobre el borde que rodea el grupo de controles.

072 <textarea> Las cajas de texto de varias líneas se crean mediante la etiqueta <textarea>. Los atributos obligatorios rows y cols establecen el número de filas y columnas iniciales de la caja, aunque los navegadores permiten modificarlo arrastrando la esquina inferior derecha.

073 <output> etiqueta se utiliza para representar el resultado de un cálculo (como uno realizado por un script).

074 <progress> etiqueta representa el progreso de finalización de una tarea.

075 <meter> etiqueta define una medida escalar dentro de un rango conocido o un valor fraccionario. Esto también se conoce como indicador.

076 <details> permite mantener oculta una parte de la página web, que el usuario puede hacer visible u ocultar de nuevo haciendo clic en un icono con forma de triángulo.

077 <summary> se incluye dentro de un elemento <details> y contiene la parte del contenido que está siempre visible y sobre la que también se puede hacer clic para mostrar el resto del contenido, como muestran los ejemplos del apartado anterior.

078 <dialog> etiqueta define un cuadro de diálogo o subventana.

079 <script> etiqueta se utiliza para definir un script del lado del cliente. (JavaScript).

080 <noscript> etiqueta define un contenido alternativo para que se muestre a los usuarios que tienen scripts deshabilitados en su navegador o que tienen un navegador que no admite scripts.
