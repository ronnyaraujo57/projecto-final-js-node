Etiquetas, Descripcion y ejemplos

# Etiqueta Descripcion ejemplos
081 <canvas> elemento se utiliza para dibujar gráficos, sobre la marcha, a través de JavaScript.

<canvas id="myCanvas" width="200" height="100"></canvas>

082 <datalist> esta etiqueta se utiliza para proporcionar una función de "autocompletar". para <<input> elementos. Los usuarios verán una lista desplegable de opciones predefinidas a medida que ingresan datos.

083 <embed> representa un punto de integración para una aplicación o contenido interactivo externo que por lo general no es HTML.

084 <video> representa un video, y sus archivos de audio y capciones asociadas, con la interfaz necesaria para reproducirlos.

085 <audio> representa un sonido o stream de audio.

086 <source> Permite a autores especificar recursos multimedia alternativos para los elementos multimedia como <video> o <audio>.

087 <map> En conjunto con <area>, define un mapa de imagen.

088 <area> En conjunto con <map>, define un mapa de imagen.

089 <keygen> representa un control de par generador de llaves.

090 <u> Este elemento representa contenido al margen, como letra pequeña (por ejemplo, descargos, restricciones legales, declaración de derechos, etc...).

091 <kdb> marca el texto que debe introducir el usuario.

092 <var> marca variables de programas y similares.

093 <title> título del documento (se muestra en la pestaña del navegador)

094 <bdi> representa un texto que debe ser aislado de sus alrededores para el formateado bidireccional del texto. Permite incrustar un fragmento de texto con una direccionalidad diferente o desconocida.

095 <ruby> representa contenidos a ser marcados con anotaciones ruby, recorridos cortos de texto presentados junto al texto. Estos son utilizados con regularidad en conjunto a lenguajes de Asia del Este, donde las anotaciones actúan como una guía para la pronunciación, como el furigana japonés.

096 <wbr> representa una oportunidad de salto de línea, es decir, un punto sugerido de envoltura donde el texto de múltiples líneas puede ser dividido para mejorar su legibilidad.

097 <rt> representa el texto de una anotación ruby.

098 <rtc> abarca anotaciones semánticas de caracteres presentados en un rubí de <rb>elementos utilizados dentro del <ruby>elemento. <rb>Los elementos pueden tener pronunciación (<rt>) y semánticas (<rtc>anotaciones de).

099 <rp> representa los paréntesis alrededor de una anotación ruby, usada para mostrar la anotación de manera alterna por los navegadores que no soporten despliegue estandar para las anotaciones.

100 <!doctype html> define que el documento esta bajo el estandar de HTML 5
